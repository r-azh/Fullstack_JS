# Rate Repository App

A React Native application for rating GitHub repositories, built with Expo.

## Features

- Rate GitHub repositories
- Sort and filter reviewed repositories
- User registration and login
- Create reviews for repositories

## Setup

1. Install dependencies:
```bash
npm install
```

2. Install additional dependencies:
```bash
npx expo install react-native-web@~0.19.6 react-dom@18.2.0 @expo/metro-runtime@~3.1.1
```

3. Start development server:
```bash
npm start
```

## Development

- Press "w" to open in browser
- Press "a" to open in Android emulator
- Press "i" to open in iOS simulator
- Scan QR code with Expo mobile app

## Linting

Run ESLint:
```bash
npm run lint
```
