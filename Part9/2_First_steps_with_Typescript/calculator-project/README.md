# Calculator Project - TypeScript Basics

Examples demonstrating TypeScript basics: type annotations, custom types, and command-line argument parsing.

## Examples

### multiplier.ts
Basic multiplier with type annotations.

### calculator.ts
Calculator with custom Operation type and error handling.

### multiplierWithArgs.ts
Multiplier with command-line argument parsing and validation.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run examples:
```bash
npm run multiply
npm run calculate
npm run multiply-args 5 2
```

## Key Concepts

- Type annotations
- Custom types (type aliases)
- Type narrowing
- Command-line argument parsing
- Error handling with unknown type
