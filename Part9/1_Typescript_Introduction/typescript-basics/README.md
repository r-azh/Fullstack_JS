# TypeScript Basics Examples

Examples demonstrating TypeScript key features from Part 9, Section 1.

## Examples

### birthdayGreeter.ts
Demonstrates type annotations:
- Function parameter types
- Return type annotation
- Type checking

### typeInference.ts
Demonstrates type inference:
- Automatic type detection
- Return type inference
- When to use explicit types

### structuralTyping.ts
Demonstrates structural typing:
- Type compatibility based on structure
- Duck typing in TypeScript
- Interface compatibility

### typeChecking.ts
Demonstrates compile-time type checking:
- Error detection before runtime
- Type safety benefits

### documentation.ts
Demonstrates code-level documentation:
- Interfaces as documentation
- Function signatures
- Type contracts

### intellisense.ts
Demonstrates IntelliSense benefits:
- Autocomplete
- Property suggestions
- Error detection

### runtimeError.ts
Demonstrates TypeScript limitations:
- No runtime type checking
- Need for validation
- Type guards

## Setup

1. Install dependencies:
```bash
npm install
```

2. Compile TypeScript:
```bash
npm run build
```

3. Run compiled JavaScript:
```bash
npm start
```

Or watch for changes:
```bash
npm run watch
```

## Key Concepts

- **Type Annotations**: Explicit type declarations
- **Type Inference**: Automatic type detection
- **Structural Typing**: Compatibility based on structure
- **Type Erasure**: Types removed at compile time
- **Static Analysis**: Compile-time error checking
