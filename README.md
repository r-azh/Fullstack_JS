# Fullstack_JS
Fullstack javascript based on https://fullstackopen.com/en/#course-contents


برای یه قسمت از فصل دو درباره پرامیس ها و فچ دو روز وقت میزاری مثل اینه که برای دستور ریکویست در پایتون دو روز وقت بزاری. یکم زیاده روی نیست. چیزی خیلی زیادتری بهت یاد نمیده و باعث میشه یادگیریت کند شه و پیشرفتت آروم باشه و خسته و ناامید شی.
یادداشت هم لازم نیست اینقدر همه چیزو کپی کنی میتونی بری همه رو تو سایت اصلیش ببینی


چیزی که بیشتر از همه اذیتم میکنه و سرعت کار رو پایین میاره نت برداری خودم و کپی همه توتوریال توی فایل  ریدمی پروژه است.  باعث میشه از کندی کار خسته شم. و از طرفی هم وقتی اینکار رو نمیکنم احساس میکنم چیزی رو از دست میدم و بعدا اگه خواستم برگردم و مرور کنم نمیتونم در حالی که الان با ابزارهای هوش مصنوعی راحت میشه خلاصه گرفت!
